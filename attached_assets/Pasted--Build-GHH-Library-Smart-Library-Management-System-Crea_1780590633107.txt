# Build GHH Library - Smart Library Management System

Create a complete production-ready mobile application named "GHH Library" using Flutter (Android & iOS) and Firebase/Supabase backend.

## Project Overview

GHH Library is a modern smart library management platform that connects library owners and students. The platform introduces a revolutionary Pay-Per-Attendance model instead of traditional monthly subscriptions.

The application should have three roles:

1. Super Admin
2. Library Owner (Admin)
3. Student

The UI should be modern, clean, premium, responsive, and easy to use.

---

# Core USP

## Pay Per Attendance System

Instead of charging for calendar days, students purchase attendance credits.

Example:

* Student purchases 30 attendance credits.
* If the student attends only 20 days and marks leave for 10 days, only 20 credits are consumed.
* Remaining credits stay available.
* Student receives full value for the purchased plan.

---

## Fair Use Policy

To prevent losses for library owners:

* Leave must be marked before a configurable cutoff time.
* Owners can allocate released seats to Daily Pass users.
* Every plan has maximum validity.

Example:

* 30 Credits Plan
* Validity = 45 Days

Unused credits expire after validity period.
